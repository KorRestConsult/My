# Илья · Life OS

Статический прототип личной операционной системы в стиле RPG-панели, премиального чеклиста и мобильной штаб-квартиры.

## Что внутри

- Главная панель персонажа
- RPG-характеристики
- Чеклист дня: утро, день, вечер
- Дневник привычек на 1000 боксов
- Проекты
- Библиотека знаний
- Мужской кабинет
- Финансы
- Резюме: 3 слота для загрузки, скачивания и печати
- Авто: паспорт машины, ТО, ремонт, сканы документов
- Здоровье: история болезни, лечение, фото/анализы, врачи и контакты
- Интеграции: дорожная карта Apple Calendar/Minical, Reminders, Health, Mist и Money Flow
- Карта жизни и наследие
- Заготовки под AI, календарь, питание/Mist и Money Flow
- Пользовательское добавление привычек, напоминаний, заметок, файлов и записей без правки кода
- Главный экран дня: привычки, дела, календарь, цели, шаги и общий прогресс
- Firebase-ready слой: локальный режим работает сразу, Firestore включается после добавления config

## Запуск

Открой `index.html` в браузере или запусти локальный сервер:

```bash
python3 -m http.server 8000
```

После этого страница будет доступна на `http://localhost:8000`.

Состояние чеклистов, боксов, заметок, загруженных файлов и записей сохраняется в `localStorage` браузера.

Важно: это первая локальная версия. Для больших файлов, синхронизации между устройствами и настоящих интеграций нужен backend или облачное хранилище.

## Firebase

Сейчас приложение работает в локальном режиме. Чтобы включить Firebase:

1. Создай Firebase project.
2. Включи Authentication с Anonymous sign-in.
3. Создай Firestore Database.
4. Возьми web app config из Firebase Console.
5. В `index.html` замени `window.LIFE_OS_FIREBASE_CONFIG=null;` на объект config.

После этого состояние будет сохраняться в Firestore в документ:

```text
users/{anonymousUserId}/lifeos/state
```

## Apple Calendar, Minical и Reminders

Minical читает системный календарь Apple, поэтому веб-сайт должен получать данные через мост:

- самый быстрый путь: Apple Shortcuts экспортирует сегодняшние события и напоминания в JSON;
- следующий шаг: Shortcut отправляет JSON в Firebase;
- после этого главный экран показывает реальные дела и календарь за сегодня.
